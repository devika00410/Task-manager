# Personal Task Manager - MERN Stack Application

A full-stack Personal Task Manager built with the MERN stack (MongoDB, Express.js, React.js, Node.js) that allows users to securely manage their daily tasks with full CRUD operations.

## Assignment Requirements Completed

### Implementation Features
- User authentication (Register/Login with JWT)
- Full CRUD operations for tasks
- MongoDB database with User and Task models
- RESTful API architecture
- Frontend-backend integration
- Error handling and validation
- Production deployment on Render

### Technical Stack
- **Frontend:** React.js with Vite
- **Backend:** Node.js with Express.js
- **Database:** MongoDB Atlas
- **Authentication:** JWT tokens
- **Deployment:** Render (Frontend + Backend)
